# Python Super Downloader

## How to run
#### Clone the source code

#### Generate lock file
Use the package manager [pipenv](https://pypi.org/project/pipenv/) to use.
```bash
pipenv lock
```

#### Install all packages
```bash
pipenv install
```

#### Run setup file to install all local packages
```bash
pipenv run runSetup
```

#### Run the program
```bash
pipenv runDefault
```
